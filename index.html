# Introduction

eslint-plugin-json-schema-validator (opens new window) is ESLint plugin that validates data using JSON Schema Validator.

This ESLint plugin validates JSON (opens new window), JSONC (opens new window), JSON5 (opens new window), YAML (opens new window), TOML (opens new window), JavaScript and Vue Custom Blocks (opens new window) with JSON Schema.
